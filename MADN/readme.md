Mensch ärgere dich nicht
=======

##Beschreibung

Das Spiel **Mensch ärgere dich nicht** als Studentenprojekt des 3. Semesters
der HTWG Konstanz im Modul "Software-Engineering". Als Programmiersprache
kommt Java zum Einsatz. Das Spiel soll eine moderne und saubere Software-Architektur 
widerspiegeln.

##Installation

Die Quellcodedateien müssen mit dem Java SDK 1.6 übersetzt werden. Eine ausführbare
.jar Datei wird zur Zeit nicht angeboten. Dies kann sich in Zukunft aber noch ändern.
